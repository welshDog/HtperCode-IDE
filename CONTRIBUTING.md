### Reporting Bugs
- Ensure the bug was not already reported by searching on GitHub under [Issues](https://github.com/welshDog/HtperCode-IDE/issues).
- If you're unable to find an open issue addressing the problem, [open a new one](https://github.com/welshDog/HtperCode-IDE/issues/new). Be sure to include a title and clear description, and as much relevant information as possible.

### Suggesting Enhancements
- Open a new issue with the [enhancement template](https://github.com/welshDog/HtperCode-IDE/issues/new?template=enhancement.md).
- Clearly describe the enhancement and why you think it should be added.

### Your First Code Contribution
1. Fork the repository
2. Create a new branch: `git checkout -b feature/your-feature-name`
3. Make your changes and commit them: `git commit -m 'Add some feature'`
4. Push to the branch: `git push origin feature/your-feature-name`
5. Open a pull request
